# Changelog

## 1.0.12
- Окно настроек в трее: хост, порт, токен, параметры whisper.
- Опция автозапуска при входе в Windows (реестр HKCU\...\Run).
- Настройки сохраняются в config.json рядом с exe (env по-прежнему в приоритете).

## 1.0.10
- Приложение работает в системном трее (без консольного окна).
- Меню трея: адрес панели, «Открыть панель», «Выход».
- Дочерние процессы (ffmpeg/whisperx/Edge) запускаются без всплывающих окон.

## 1.0.5
- Кнопка удаления записи (с подтверждением); удаляет MP3 и его транскрипты.

## 1.0.3
- Мультизапись: несколько параллельных записей в одном приложении.
- Каждая сессия имеет свой id, статус, индикатор уровня и кнопку «Стоп».
- Эндпоинты: `/start` возвращает id, `/stop` принимает id, новый `/sessions`.
- Панель показывает список активных записей.

## 1.0.0
- Запись аудио Jitsi Meet в MP3 (headless Edge через CDP).
- REST-управление и веб-панель (start/stop/status, список записей, скачивание).
- Индикатор уровня громкости во время записи.
- Транскрибация записей через whisperx с скачиванием результатов.
- Сервер по умолчанию на `0.0.0.0:9999`, доступен по локальной сети.
- Эндпоинт `/version`, версия в панели и в баннере запуска.
